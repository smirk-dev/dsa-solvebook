📄 Document 1: Project Brief (The Analyst)
Project Name: dsa-solvebook

Vision: A self-sustaining, automated "Proof of Work" portfolio that tracks and displays LeetCode progress with zero manual entry.

Primary Objective: Create a high-fidelity, searchable "Solvebook" that reflects real-time LeetCode activity.

Target Audience: The user (for tracking) and potential hiring managers (as a technical portfolio).

Core Value Prop: Decouples the "doing" (LeetCode) from the "displaying" (Static Site) via a Git-based intermediary
