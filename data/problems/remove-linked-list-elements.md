## Problem

**Example 1:**
    
    
    **Input:** head = [1,2,6,3,4,5,6], val = 6

**Example 2:**
    
    
    **Input:** head = [], val = 1
    **Output:** []
    

**Example 3:**

    **Output:** []
    

 

**Constraints:**

  * The number of nodes in the list is in the range `[0, 104]`.
  * `1 <= Node.val <= 50`
